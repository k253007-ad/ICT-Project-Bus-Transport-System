# Bus Transport System

A clean, simple, and functional bus registeration system. Built with HTML, CSS, and JavaScript.

## Features

- ✅ Search bus stops
- ✅ View available buses at each stop
- ✅ See complete bus routes
- ✅ Check seat availability
- ✅ Mobile responsive
- ✅ GitHub Pages ready
